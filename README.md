# Blinkit Business Intelligence Dashboard

## Project Overview
End-to-end business analytics project analyzing Blinkit's
operations across 80,000+ records covering orders, customers,
products and inventory for March 2023 to November 2024.

## Tools Used
- Excel — Data cleaning and feature engineering
- SQL (PostgreSQL) — Business queries and data analysis
- Python (Pandas, Matplotlib, Seaborn) — Statistical analysis and visualization
- Power BI — Interactive dashboard

## Dataset
5 interconnected tables:
- Orders (5,000 records)
- Products (268 records)
- Order Items (5,000 records)
- Customers (2,500 records)
- Inventory (75,172 records)

## Key Insights
1. Revenue peaked during Diwali (Oct 2024) confirming festive seasonality
2. 30% of orders face delivery delays — 9.7% significantly delayed
3. Dairy & Breakfast leads revenue; Frozen Food leads margins
4. 24% inactive customers represent major reactivation opportunity
5. Household Care has highest inventory damage rate

## Business Recommendations
1. Boost Frozen Food visibility to improve profitability
2. Reactivate 24% inactive customers through targeted campaigns
3. Improve packaging for high damage categories

## Project Structure
